# Hospitality-Analytics
# Business Problem
The hotel management wanted to understand revenue performance and identify opportunities to improve profitability.

# Approach
1. Analyzed 100,000+ hotel booking records.
2. Calculated key hospitality metrics:
   ADR (Average Daily Rate) = Total Room Revenue ÷ Rooms Sold
   RevPAR (Revenue Per Available Room) = Total Room Revenue ÷ Available Rooms
3. Built interactive Power BI dashboards to analyze revenue, occupancy, and booking trends.

# Tools Used
Power BI
Excel
SQL

# Outcome
The analysis identified revenue improvement opportunities and provided data-driven insights to support pricing and occupancy decisions.
